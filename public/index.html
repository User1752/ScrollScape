Sobre

Este é um MVP inspirado no modelo de “sources/extensions” do Tachiyomi/Mihon, mas implementado para Web. Instalar fontes significa executar código JavaScript no servidor — use apenas repositórios confiáveis.

Dica: Se quiseres, eu posso adaptar isto para suportar biblioteca local (pastas/CBZ) ou autenticação para serviços oficiais.
